# 다양한 데이터 기술을 적절하게 사용하는 방법

## 1. 데이터(Data)

- 소프트웨어 엔지니어로 경력을 시작할 때 프로그래밍 언어, 프레임워크, 개발 도구를 배우는 데만 매몰되기 쉽다.
- 하지만 실제 엔지니어링 환경에서는 다양한 형태의 데이터를 다루는 데 대부분의 시간을 보내게 되며, 데이터는 우리가 구현하는 모든 비즈니스 로직의 근간이다.
- 애플리케이션은 데이터를 수집하고, 저장하고, 변환하고, 최종적으로 사용자에게 의미 있는 방식으로 되돌려주는 생명 주기를 중심으로 모델링된다.
- 시스템 개발에서 데이터를 다룬다는 것은 원시 자료형(Primitive Type)의 이해부터 시작하여 적절한 저장소(Storage) 선택, 효율적인 데이터 모델 설계, 조회(Query) 최적화, 데이터 무결성(Integrity) 보장, 시간에 따른 스키마 진화 관리 등 훨씬 더 넓고 깊은 엔지니어링 영역을 포함한다.
